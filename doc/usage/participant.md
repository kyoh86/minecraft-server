# Minecraft の使い方（for プレイヤー）

## ログイン方法

ログインにはDiscordでの操作が必要になるため、専用Discordサーバーに参加する必要があります。

1. サーバー接続を試みると、`limbo` サーバーに接続されます。
1. チャットでログイン用の一時コードを受け取れるので、 `t` キーなどでメッセージ履歴を表示します。
1. コードをクリックするとコピーされるので、専用Discordサーバーで Discord で `/mc link code:<表示されたコード>` を実行します。

## ワールド構成

このサーバーは複数のワールドで構成されています。

- `mainhall`: 移動ハブ（Adventure / Peaceful / PvP無効 / モブ湧き無効）
- `residence`: 生活・拠点建築（Survival / Normal）
- `resource`: 資源採取（Survival / Hard）
- `factory`: 装置・自動化（Survival / Normal）

ワールド間の移動には、専用のゲートを使用してください。
`residence` / `resource` / `factory` には `mainhall` へ戻るゲートが用意されているので、ワールド間の移動は `mainhall` のゲートを経由します。

## ゲート周辺の保護

ゲートへの出入りを保護するため、以下の領域は建築・破壊・爆発など抑止しています。

- ワールド: `residence`, `resource`, `factory`
- `x,z=-32..32`
- `y=-64..319`

## ClickMobs

ワールド間のモブ移動をサポートするため、以下の領域で全モブの捕獲が有効化されています。
Shift＋右クリックでモブを捕獲できます。また、捕獲して得たHead blockを設置すると、捕獲したモブをその場に解き放つことができます。

- ワールド: `residence` / `resource` / `factory`
- `x,z=-64..64`
- `y=-64..319`

## ワールドボーダー

一部のワールドには、ワールドボーダーを設定しており、それ以上の座標にはアクセスできなくなっています。

- ワールド: `residence` / `factory`
- 中心 `0,0`、直径 `10000`
