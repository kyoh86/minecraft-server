# スイッチ向け: マインクラフト サーバーに入るまでのガイドライン

ウインドウズ環境でプレイしたい方は [ウインドウズ環境向けガイドライン](../entry-windows) へ。

## サーバー接続までの手順

- Discordの #マインクラフト-server チャンネルに参加
- マインクラフト（統合版）の購入
- Xbox ゲーマーアカウントの用意
- マインクラフトで Xbox ゲーマーアカウントにログイン
- サーバーへの接続設定

手順ごとに説明します。

## Discordの #マインクラフト-server チャンネルに参加

山田から招待を送りますので、コンタクトをとってください。

## マインクラフトの購入

スイッチ用のソフトを購入してください
https://www.minecraft.net/ja-jp/store/minecraft-switch

## Xbox ゲーマー アカウントの用意

Xbox ゲーマーアカウントは、スマホやパソコンのブラウザで [Xbox live](https://www.xbox.com/ja-JP/live) から作ることができます。

1. すでに持っている可能性があるので、サインインを試してみましょう。
2. 持っていない場合はアカウントを作成しましょう。
3. アカウントが用意できたら、 [Xbox プロファイル](https://account.xbox.com/ja-jp/Profile) ページにアクセスしてみましょう。ログインできていれば、自分のアカウント名が表示されます。

## マインクラフトで Xbox ゲーマーアカウントにログイン

1. スイッチでマインクラフトを起動します。
2. オープニング画面の左下の「ニンテンドーアカウントでログイン」で、ニンテンドーのアカウントでログインします。
    - (画像割愛)
3. 左下の「マイクロソフトアカウントでログイン」が表示されるので、マイクロソフトのアカウントでログインします。
    - ![](https://i.gyazo.com/d651b9cb0dad4efc9ad848b73ce321b9.jpg)
4. ダイアログが表示されるので、その案内に従って、スマホやパソコンで表示されたURLにアクセスします。
    - ![](https://i.gyazo.com/a02f84ba109ac1a76e7336745653d2b9.jpg)
    - コードの入力を求められるので、表示されたコードを入力しましょう。
5. 確認のため、スマホやパソコンのブラウザで [Xbox プロファイル](https://account.xbox.com/ja-jp/Profile) ページにアクセスしてみましょう。
    - 大きく表示されるのが Xboxゲーマータグ（アカウント名）です。![](https://i.gyazo.com/af7dfdbc8b0899267a1121cc5956539b.png)
    - 正常にログインできていれば、マインクラフトの左下にがXboxゲーマータグが表示されます。  
![](https://i.gyazo.com/cf14bc925a7a67dc5134eb8c49d72aa3.jpg)

## サーバーに接続する

サーバーに接続するには、

1. スイッチ本体の設定変更
2. マインクラフトで接続先を変える

の2ステップが必要です。

### スイッチ本体の設定変更

スイッチは「ライフボート」といった公式特集サーバーにしか入れないようになっています。マインクラフトサーバーに参加するには、スイッチ本体の設定変更が必要です。

#### 注意点

- **※1.** この設定は使用しているインターネット環境が変わるたびに設定する必要があります。**別のインターネット環境（Wi-Fi環境）で遊ぶときは、またこの設定を行う必要があります**
- **※2.** **必ず大人の許可・監督の下で行ってください。**

1. スイッチ本体の「設定」を開きます。

2. 「インターネット」の一番上にある「インターネット設定」を開きます。
    - ![](https://i.gyazo.com/6229a520a63929689f0efd349384aa21.jpg)
3. 接続できるネットワークの一覧が出てきます。チェックが付いているネットワークを選んで、「設定の変更」に進みます。
    - ![](https://i.gyazo.com/46f57eba8c1e01a4ba6173991554ca3e.jpg)
4. 「DNS設定」という項目を、以下のように変更します。
    - ![](https://i.gyazo.com/0587bf53852cb1ba6738ba3a337dec50.jpg)
    - DNS設定: 手動
    - 優先DNS: `3.114.80.244`
    - 代替DNS: `008.008.008.008`

終わったら「保存する」を押してください。

### マインクラフトでこのサーバーに接続する

マインクラフトの起動画面から、右上の「サーバー」というタブを開き、特集サーバーの次のうちいずれかに参加してください。

- The Hive
- GALAXITE
- Lifeboat

すると、 `Server List` という画面が表示されます。

![](https://i.gyazo.com/8c83c02060a61936369a8c4d753d0ded.jpg)

> `Server List` という画面が表示されず、いきなりゲーム画面に入ってしまった場合は、スイッチ本体の設定変更に失敗しているかもしれません。もう一度設定を確認してください。

`Connect to a Server` を選んでください。

サーバーの接続先を設定する画面が開くので、以下の通り入力してください。

- `Server Address`: `mc.kyoh86.dev`
- `Server Port`: `19132`
- `Display Name on Server List`: （自分にとってわかりやすい名前なら何でもいい）
- `Add to Server List`: オン（右側にトグルして緑色になっている）

もとの `Server List` という画面に戻るので、登録した名前（`Display Name`）のボタンで接続できます。

### Discordで受け取ったコードを入力する

初めて参加すると、画面にメッセージとともに**5桁くらいの数字**が表示されるので、これをDiscordでコマンドに渡す必要があります。
![](https://i.gyazo.com/4943e24ee54fdecf85cfd71c16c72e58.png)
Discordのメッセージ欄に `/link` と入力すると、コマンド一覧が表示されるので、 `Minecraft Discord Integ`の `/link`コマンドを選びます。
![](https://i.gyazo.com/e12a41568dbe16c39ef93b8c19f81a6d.png)
`code`を入力するよう促されるので、先ほど受け取ったコードを入力してください。
![](https://i.gyazo.com/7f8909e41355ee5c0352883474958948.png)
Enterキーで送信すると、ワールドに入れるようになります。Switch側で繋ぎ直してください。
